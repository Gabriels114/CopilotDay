# Challenge 00 - Prerequisites - Ready, Set, GO!

Este challenge deja listo el entorno para trabajar el resto del `WhatTheHack 071 - GitHub Copilot`.

## Objetivo

Tener una estación de trabajo funcional para:

- usar GitHub con una cuenta personal activa;
- trabajar en Visual Studio Code;
- usar GitHub Copilot dentro del editor.

## Contexto

La guía original menciona prerrequisitos comunes de hacks orientados a Azure, pero para este hack los mínimos reales son:

- cuenta de GitHub;
- Visual Studio Code instalado;
- acceso a GitHub Copilot;
- extensión de GitHub Copilot funcionando en VS Code.

## Checklist

Marca cada punto cuando quede validado.

- [ ] Tengo una cuenta de GitHub y puedo iniciar sesión.
- [ ] Mi correo de GitHub está verificado.
- [ ] Tengo Visual Studio Code instalado.
- [ ] Puedo abrir este repositorio en VS Code sin problemas.
- [ ] Tengo acceso a GitHub Copilot en mi cuenta personal u organización.
- [ ] La extensión de GitHub Copilot está instalada y habilitada en VS Code.
- [ ] Veo sugerencias inline o puedo abrir Copilot Chat desde VS Code.

## Pasos

### 1. Crear o validar cuenta de GitHub

1. Ve a `https://github.com/`.
2. Crea tu cuenta o inicia sesión.
3. Verifica tu correo si todavía no lo has hecho.
4. Recomendado: activa `2FA`.

### 2. Instalar Visual Studio Code

1. Descarga VS Code desde la web oficial.
2. Instálalo para tu sistema operativo.
3. Abre este repositorio como carpeta de trabajo.

### 3. Activar GitHub Copilot

1. En VS Code, inicia sesión con tu cuenta de GitHub.
2. Instala o habilita la extensión de GitHub Copilot si no aparece activa.
3. Si no tienes plan activo, revisa las opciones actuales de acceso a Copilot.
4. Confirma que el panel de chat y/o las sugerencias inline están disponibles.

### 4. Validación mínima

Haz una prueba simple:

1. Crea un archivo temporal, por ejemplo `scratch.js`.
2. Escribe un comentario como `// create a function that sums two numbers`.
3. Espera una sugerencia inline o abre Copilot Chat y pide:
   `Create a sum function in JavaScript`.
4. Si ves una respuesta útil en el editor, el entorno está listo.

## Evidencia sugerida

Para dejar constancia en el repo o en tu entrega del hack:

- captura de sesión iniciada en GitHub;
- captura de VS Code con Copilot Chat visible;
- captura de una sugerencia inline o de una respuesta de Copilot;
- nota corta sobre si usas `Copilot Free`, plan individual u organización.

## Criterio de éxito

Considera completado este reto cuando:

- puedes iniciar sesión en GitHub;
- puedes abrir el repo en VS Code;
- Copilot responde en el editor o en el chat.

## Notas actuales

- La documentación actual de GitHub indica que `Copilot Free` puede activarse desde VS Code o GitHub para muchos usuarios individuales.
- La documentación de GitHub también indica que, desde el `20 de abril de 2026`, las nuevas altas para `Copilot Pro`, `Copilot Pro+` y planes de estudiante están temporalmente pausadas. Si no tienes acceso ya activo, conviene revisar el estado vigente antes de continuar.
