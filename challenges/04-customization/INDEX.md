## 💡 Tips de Navegación

### En VS Code
- Usa `⌘K ⌘O` (macOS) o `Ctrl+K Ctrl+O` para abrir una vista lateral con todos los documentos
- Usa `⌘F` para buscar en documentos

### En GitHub Web
- Los documentos están en `challenges/04-customization/`
- Haz clic en el filename para ver en raw

### Para imprimir/capturar
- Usa "Print" (⌘P) → "Save as PDF"
- Captura pantallas de los pasos de TEST-INSTRUCTIONS.md
